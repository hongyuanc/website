Movie Recommendation System

Technology Used: Python, Pandas, TKinter, Scikit-learn

My first project. This program feeds in data from two existing csv files (movie databases), cleans it, then starts to combine and vectorize features. The key column is then vectorized using CountVectorizer from Scikit-Learn, then the cosine similarity is computed between these vectors to measure the similarity. Then it gives recommendations.
